📌 About SFTSV-DB

SFTSV-DB is an open-access genomic surveillance platform for Severe fever with thrombocytopenia syndrome virus (SFTSV), a tick-borne bunyavirus with high mortality in East Asia. The platform integrates public genomic data from NCBI, performs standardized phylogenetic analysis using Nextstrain (Augur + Auspice), and provides interactive visualization for real-time monitoring.

Current version: S segment only (M/L segments and reassortment analysis in development).
